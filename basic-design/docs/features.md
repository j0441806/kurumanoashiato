# 機能一覧

## ユースケースごとの機能一覧

### TOP画面

- 会員登録する
- ログインする

- あしあとを閲覧する
- あしあとを検索する
    - 公開されたあしあとを閲覧・検索できる

## マイページ画面

- プロフィールを更新する
- パスワードを変更する

- あしあとを閲覧する
- あしあとを検索する
- あしあとを編集する
    - コメント・写真を登録できる

- あしあとを記録する
    - 内部の処理・・・
        - スマホの位置情報を取得（座標情報を取得）
        - 取得した情報を地図上に表示
        - 移動した軌跡を地図上に描画
        - 記録を保存したとき、軌跡を保存する

- 過去のデータを確認する
    - 走行距離を確認する
    - 走行時間を確認する

- お気に入りスポットを登録・閲覧する
- お気に入りルートを登録・閲覧する
- アルバムを登録・閲覧する


## 管理画面
- ユーザー情報を閲覧・検索・編集できる
- あしあとの情報を閲覧・検索・編集できる
